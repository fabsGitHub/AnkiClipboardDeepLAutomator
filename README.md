# Anki Translator

A script to translate selected text and create Anki flashcards automatically.

## Prerequisites

- [Anki](https://apps.ankiweb.net/)
- [AnkiConnect Plugin](https://foosoft.net/projects/anki-connect/)
- Python 3.x
- Required Python packages: `pyperclip`, `requests`, `deepl`, `pynput`, `python-dotenv`

## Installation

1. Clone this repository or download the script files.
2. Install the required Python packages:

   ```bash
   pip install pyperclip requests deepl pynput python-dotenv
   ```
